# Java Object-Oriented Programming Studies

This repository contains exercises and projects developed during my Object-Oriented Programming (OOP) studies in Java.  
The goal is to practice fundamental programming concepts and strengthen software development skills through practical implementation.

## Topics Covered

The repository includes examples and exercises related to:

- Classes and Objects
- Encapsulation
- Constructors
- Inheritance
- Polymorphism
- Method Overloading and Overriding
- Abstraction
- Data Structures
- File manipulation
- Basic algorithms

## Project Structure

The repository is organized by exercises and small projects developed during the course.
